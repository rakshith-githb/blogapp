# MERN Blog Application

A full-stack blog platform built using MERN stack.

## Features

- User Authentication (JWT)
- Role-based Access (User, Author, Admin)
- Create, Edit, Delete Articles
- Image Upload (Multer)
- Comments System
- Admin Dashboard
- Soft Delete & Restore (Trash system)

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Compass

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
